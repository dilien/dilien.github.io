I'm a student with a strong passion for software development. This website showcases a collection of projects that I've found particularly educational or interesting. The projects span different programming languages, including Python, JavaScript, and C#, and some are even playable directly in the browser.

In this full-stack project, you control a fort equipped with artillery weapons and aim to destroy the opposing player's base. It uses TypeScript for both the frontend and backend. Since the game is a replica of the existing game REDCON, I focused solely on the programming aspects, with no need to work on the graphics. This project is still a work in progress.

In this python project, I made a multiplayer top down shooter for me and my friends to play. It used realtime shadows to make the game more realistic and fun. Since the client python code was just a text file, it was easily editable so I had to put a lot of effort into learning about and building a robust anticheat.


I made a 3D renderer in javascript, without using any external libaries or WebGL. As a result, it does run quite slowly (and doesnt have textures) but it is playable. I used it to make a simple copy of minecraft, which you can play by clicking the link.

This project is a precursor to my full 3D renderer and features a basic raycasting engine. Initially, it was just a maze to navigate, but I later added enemies that would attack using an efficient group A* pathfinding system, which only requires a single calculation for the entire group.

This Python program is an AI for Worldle, but not in the traditional sense. Instead of guessing letters, it dynamically changes the word and provides hints that offer the least amount of information, using probability. This makes the game much more challenging, as the AI constantly aims to minimize the clues it gives, keeping you on your toes.
