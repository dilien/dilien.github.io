I am Dillon Millar, a student with a passion for software development. This website is a list of all the projects I felt I learnt a lot from, or are interesting. They vary in language from python to javascript to C#. Some of them are even playable in the browser.

I made a 3D renderer in javascript, without using any external libaries or WebGL. As a result, it does run quite slowly (and doesnt have textures) but it is playable. I used it to make a simple copy of minecraft, which you can play by clicking the link.

playable in the browser

In this python project, I made a multiplayer top down shooter for me and my friends to play. It used realtime shadows to make the game more realistic and fun. Since the client python code was just a text file, it was easily editable so I had to put a lot of effort into learning about and building a robust anticheat.

In this fullstack project, you control a fort of artilery weapons and try to destroy the other players base. It uses typescript for both the frontend and backend. Since the game is a carbon copy of an existing game, REDCON, it left me to only have to do the programming rather than any of the graphics work. This project is still ongoing.

In this precursor to my full 3D renderer, it was a simple raycasting engine. At first, it was simply mazes you had to walk through but I added enemies that would attack using a efficent group A* Pathfinding system, requiring only one call for the whole group.

This python program is a AI for worldle but not in the way you might think. Instead of guessing letters, it dynamically changes the word and it gives the hints that give the player the least information, using probability. This makes it much harder to win, as the AI is always trying to give you the least information
